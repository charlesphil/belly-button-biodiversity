Definition of Terms

OTU: Operational Taxonomic Unit, a unit that refers to either an individual, species, genus, or class. Usually clustered with similar genetic sequences.

bbtype: Inner or outer belly button type.

wfreq: How frequently subject washes belly button each week.

Data Visualizations
About This Dashboard

This static dashboard is an exercise in the loading and manipulation of data to use in web-based visualizations. The dataset comes from the Belly Button Biodiversity dataset from The Public Science Lab at North Carolina State University.

This project uses the Bootswatch Morph theme, based on Bootstrap 5, for constructing and styling the dashboard. Functionality and visualization is provided by the D3 (version 7) and plotly.js (version 2.9.0) JavaScript libraries.

For the source code, please visit this project's GitHub Repository.
